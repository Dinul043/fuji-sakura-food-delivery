# Components Folder

This folder contains reusable React components used throughout the application.

## Purpose
- Reusable UI components
- Layout components
- Form components
- Feature-specific components

## Structure
```
components/
├── ui/           # Basic UI components (buttons, inputs, cards)
├── layout/       # Layout components (header, footer, sidebar)
├── forms/        # Form-related components
└── features/     # Feature-specific components
```

## Examples
- `ui/Button.tsx` - Reusable button component
- `layout/Header.tsx` - App header component
- `forms/SearchInput.tsx` - Search input component
- `features/RestaurantCard.tsx` - Restaurant card component

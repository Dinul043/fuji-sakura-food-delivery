# Utils Folder

This folder contains utility functions and helper methods used throughout the application.

## Purpose
- Pure functions for data manipulation
- API helpers and formatters
- Common validation functions
- Date/time utilities
- String manipulation helpers

## Examples
- `formatCurrency.ts` - Format prices and currency
- `validateEmail.ts` - Email validation logic
- `debounce.ts` - Debounce utility for search
- `localStorage.ts` - LocalStorage wrapper functions
